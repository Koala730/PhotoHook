# 使用前必读

## 缺少 gradle-wrapper.jar

由于文件大小限制，本项目没有包含 `gradle/wrapper/gradle-wrapper.jar`。

### 解决方法（任选其一）

#### 方法一：用 Android Studio 生成（最简单）

1. 用 Android Studio 打开项目
2. 等待 Gradle 同步完成
3. Android Studio 会自动下载并生成 `gradle-wrapper.jar`

#### 方法二：从其他 Android 项目复制

1. 找一个现成的 Android 项目
2. 复制 `gradle/wrapper/gradle-wrapper.jar` 到本项目的 `gradle/wrapper/` 目录下

#### 方法三：只使用 GitHub Actions（推荐）

本项目已配置好 GitHub Actions，您只需要：
1. 把整个项目上传到 GitHub
2. Actions 会自动处理（不需要本地有 gradle-wrapper.jar）
3. 直接下载编译好的 APK

## 推荐方案

**直接用 GitHub Actions 在线编译！** 不用管 gradle-wrapper.jar，Actions 环境会自动处理。

步骤：
1. 在 GitHub 创建一个新仓库
2. 把 PhotoHook-Complete 文件夹里的所有内容上传
3. 等 2-3 分钟，Actions 自动编译完成
4. 下载 app-debug.apk
